Nexu v0.1.7 — full changelog 🚀

**Highlights**

1. 🔧 **Rock-solid startup** — No more white screen, “can’t open”, or channels stuck on “connecting”. Resilient to port conflicts, upgrades, and force-quit recovery.
2. 🔑 **OpenAI Codex OAuth** — ChatGPT Plus/Pro login for GPT models (#537)
3. 🔑 **MiniMax OAuth** — One-click model access + refined provider UI (#549)
4. 🚀 **Z.AI Coding Plan** — GLM quick-setup with Global/CN toggle (#537)
5. 🛡️ **Self-healing services** — Launchd rebuilt: plist drift auto-detected, stale services cleaned on startup, Stop/Start reliable

**Also**

6. 📖 **WeChat setup guide** — English channel docs (#548)  
7. 📖 **Discord guide refresh** — New screenshots, simpler permissions (#528)

🐛 **Fixes (summary):** White screen after upgrade; channels connecting forever on port clash; Stop ignored by KeepAlive; Stop/Start greyed out; stale web port after force-quit; **v0.1.6 data path** — sessions/skills/channels migrated back; OAuth disconnect now clears models; expired OAuth hidden from selector; skill detail Back tab; Skills page tab in URL; more.
